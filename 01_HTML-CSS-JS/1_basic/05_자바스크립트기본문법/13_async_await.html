async await

      Javascript에서 비동기 코드를 쉽게 작성하기 위한 문법
      Promise를 동기처럼 쓸 수 있다.
      
      async function 함수명() {
        const 결과 = await Promise_함수();
      }
      
      const 함수명 = async () => {
        const 결과 = await Promise_함수();
      }
      
      - async : 함수 앞에 붙이면 무조검 Promise를 반환
      - await : Promise가 끝날 때 까지 기다림 (async를 앞에 선언한 함수 안에서만 사용 가능)
